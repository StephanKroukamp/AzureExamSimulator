# Microsoft Exam Simulator

Practice for Microsoft certifications with a sleek, modern web UI. Covers 57 exams across Azure, Security, Data & AI, Microsoft 365, Power Platform, Dynamics 365, GitHub, and Office.

## Prerequisites

- [.NET 10 SDK](https://dotnet.microsoft.com/download/dotnet/10.0)

## Getting Started

```bash
git clone https://github.com/StephanKroukamp/AzureExamSimulator.git
cd AzureExamSimulator
dotnet run
```

A browser window will open automatically. If it doesn't, navigate to **http://localhost:5199**.

## Features

- **57 Exams** — covering the full Microsoft certification landscape
- **Search & Filter** — find exams by code or name, browse by category
- **Practice Mode** — instant feedback and explanations after each question
- **Timed Exam Mode** — simulate real exam conditions with a countdown timer
- **Configurable** — choose question count and time limit per session
- **Topic Breakdown** — see your performance across exam domains
- **Dark / Light Theme** — toggle via the button in the header
